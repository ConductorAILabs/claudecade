  §1   INSTALL                                                          p.01
══════════════════════════════════════════════════════════════════════════════




  Two paths.  Pick one.







  ──  A.   WITH CLAUDE CODE   ───────────────────────────────────  recommended




  1.   Add this to  ~/.claude/settings.json


           {
             "mcpServers": {
               "claudcade": {
                 "type": "http",
                 "url":  "https://starlit-macaron-113a83.netlify.app/api/mcp"
               }
             }
           }




  2.   Open Claude inside a tmux session


           tmux
           claude




  3.   Tell Claude to install it


           set me up with Claudcade




  4.   Play any time


           /claudcade







  ──  B.   WITHOUT CLAUDE CODE   ────────────────────────────────────  manual




  1.   Download and unzip the games


           curl -L https://starlit-macaron-113a83.netlify.app/claudcade.zip \
                -o /tmp/claudcade.zip
           unzip -q /tmp/claudcade.zip -d ~/claudecade




  2.   Launch the arcade


           cd ~/claudecade && python3 claudcade.py




  3.   Or play one game directly


           python3 ctype.py            ·   space shooter
           python3 claudtra.py         ·   run & gun
           python3 fight.py            ·   1v1 fighting
           python3 superclaudio.py     ·   side-scroll platformer
           python3 claudturismo.py     ·   top-down racing
           python3 claudemon.py        ·   creature RPG
           python3 finalclaudesy.py    ·   epic JRPG

       Python 3.10+        ·       built-in curses        ·       no pip install

  ▸  Common questions. If yours isn't here, file an issue on GitHub.

◆ ─ ◇ ─ ◆ ─ ◇ ◆ ─ ◇ ─ ◆ ─ ◇ ◆ ─ ◇ ─ ◆ ─ ◇ ◆ ─ ◇ ─ ◆ ─ ◇ ◆ ─ ◇ ─ ◆ ─ ◇ ◆ ─ ◇ ─ 

┌─[ Q1 ]─ What is Claudcade? ────────────────────────────────────────────────┐
│                                                                            │
│    A terminal arcade you launch from inside Claude Code.                   │
│    Seven games written in pure Python curses, scores backed                │
│    by a real Postgres-powered global leaderboard.                          │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q2 ]─ Do I need Claude Code to play? ────────────────────────────────────┐
│                                                                            │
│    No.  Section §1.2 (manual install) downloads a zip and                  │
│    you run  python3 claudcade.py  directly. Claude Code                    │
│    just makes the setup one prompt instead of three commands.              │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q3 ]─ Is it free? ───────────────────────────────────────────────────────┐
│                                                                            │
│    Yes. MIT-licensed source, no fees, no telemetry beyond                  │
│    the optional score submission to the global leaderboard.                │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q4 ]─ Do my scores actually save? ───────────────────────────────────────┐
│                                                                            │
│    Yes. Successful submissions land in the leaderboard DB.                 │
│    Failed submissions (offline / server down) cache locally                │
│    as personal bests in  ~/.claudcade_scores.json  and you                 │
│    can keep playing — they do not block the game loop.                     │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q5 ]─ Why does it use tmux? ─────────────────────────────────────────────┐
│                                                                            │
│    Just for full-screen playability — each game takes over                 │
│    the terminal via curses. You can run individual games                   │
│    without tmux too:  python3 ctype.py  works fine alone.                  │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q6 ]─ Does it work on Windows? ──────────────────────────────────────────┐
│                                                                            │
│    Via WSL, yes. Native Windows curses is patchy; WSL                      │
│    Ubuntu gives you a real terminal with full color and                    │
│    box-drawing character support.                                          │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q7 ]─ Can I add my own game? ────────────────────────────────────────────┐
│                                                                            │
│    Yes. Copy  claudcade_engine.py  into your project, use                  │
│    any of the 4 included games as a template, or ask                       │
│    Claude:  > scaffold me a Claudecade  called .              │
│    See §2 for the engine API.                                              │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─[ Q8 ]─ Why ASCII? ────────────────────────────────────────────────────────┐
│                                                                            │
│    It runs anywhere with a monospace terminal, loads                       │
│    instantly, and the constraint forces interesting design                 │
│    decisions. Plus there is no asset pipeline to maintain.                 │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

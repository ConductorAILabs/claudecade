INSTALL

SETUP GUIDE

01
Add the MCP to Claude Code

Open ~/.claude/settings.json and add the Claudecade entry inside mcpServers:

~/.claude/settings.json
"mcpServers": {
  "claudcade": {
    "type": "http",
    "url": "https://starlit-macaron-113a83.netlify.app/api/mcp"
  }
}
02
Start a tmux session

tmux lets Claude Code and games run side by side. Open your terminal:

terminal
$tmux

No tmux? brew install tmux on macOS  ·  sudo apt install tmux on Linux

03
Let Claude install everything

Inside tmux, launch Claude Code and say the magic words. Claude runs every command — no git, no manual steps.

tmux → Claude Code
$claude
>set me up with Claudecade
↳ downloads games, installs /claudcade command
04
Play — any time

Whenever you want to play — tmux open, Claude Code running:

Claude Code
>/claudcade
↳ arcade opens full-screen  ·  arrows to pick  ·  enter to launch

Your first game-over registers you on the global leaderboard. No account needed.
